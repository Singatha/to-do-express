# To do list

A to do list REST API built with expressjs and sqlite.


## Run Locally

Clone the project

```bash
  git clone https://github.com/Singatha/to-do-express.git
```

Go to the project directory

```bash
  cd to-do-express
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node app.js
```
